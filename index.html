Plotbras

Some simple economic and social indicators about Brazil. Data are retrieved from web datasets provided by well known organizations, such as United Nations Statistics Division Data, World Health Organization, World Bank... and so on

Download this project as a .zip file Download this project as a tar.gz file

Em (perene) construção. Moto:

O que não se mede não se discute

Propósito

Divulgar gráficos simples sobre indicadores _simples_ de dados públicos divulgados por organizaçõe renomadas, tais como União das Nações Unidas, Banco Mundial, FAO, FMI, Banco central do Brasil, IBGE, etc.

Uma fonte agregadora de dados bastante rica é o Quandl (quandl.com)

Os códigos dos gráficos estarão no repositório ligado a esta página e estão escritos em R.

As figuras geradas pelos códigos aqui depositados serão publicadas aqui.
